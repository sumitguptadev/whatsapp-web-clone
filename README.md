# 💬 WhatsApp Web Clone

A modern and responsive **WhatsApp Web Clone** built using **HTML, CSS, and JavaScript** with smooth animations, interactive hover effects, and a clean UI design.

This project was created to improve my frontend development skills and practice building real-world web interfaces.

---

## 🚀 Features

- ✨ Smooth UI animations
- 🎨 Modern WhatsApp-inspired design
- 🖱️ Interactive hover effects
- 📱 Responsive layout
- ⚡ Clean and lightweight frontend project
- 💻 Beginner-friendly code structure

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript**
